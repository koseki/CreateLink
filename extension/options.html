Variables

Create Link recognizes some Make Link compatible variables.
%text%
Selected text or page title. All newlines are converted to " ".
%text_n%
Selected text or page title. All newlines are copied as they are.
%text_br%
Selected text or page title. All newlines are converted to "<br />\n".
%title%
Page title.
%newline%
Newline(\n).
%htmlEscapedText%
Selected text or page title. Some unsafe characters (&<>'") are converted to corresponding entity reference.
%input%
Popups a dialog to input text.

Key Configuration

Page reload is required when you change the key configuration.

Format: [CMSARU]+{key},[CMSARU]+{key},...

C
Ctrl key
M
Command / Meta key
S
Shift key
A
Alt / Option key
R
Only when selecting text or input field has forcus.
U
Not 'R'.

Example

CUC
Ctrl-C only when selecting nothing.
MS1,CAX
Command-Shift-1 or Ctrl-Option-X
